# AI Agent 通识入门 — 不写代码，也能看懂 AI Agent

> 基于真实项目《从 ClawBot 到 OpenClaw》，面向零编程基础学习者的 AI Agent 科普课程。

## 课程概览

- **课程性质：** 入门级通识课程
- **总课时：** 6 次课 × 3 小时 = 18 学时
- **目标受众：** 零编程基础（产品经理、运营、管理者、AI 爱好者）
- **核心理念：** 不要求写代码，用类比和案例理解 AI Agent

## 课程大纲

| 课次 | 主题 | 核心类比 |
|------|------|----------|
| 第 1 课 | AI Agent 是什么？从"忘事的助手"说起 | 私人秘书 / 四层记忆 |
| 第 2 课 | AI 的"工具箱"—— 一个模型做不了所有事 | 专科医院 / 分诊台 |
| 第 3 课 | AI 团队协作 —— 从独行侠到"AI 公司" | 公司组织架构 / OA 系统 |
| 第 4 课 | AI 能做什么？写代码、写文章、做决策 | 麦当劳 SOP / 建房子 |
| 第 5 课 | AI 的安全边界 —— 从量化交易看风险管理 | 汽车安全系统 / 体检报告 |
| 第 6 课 | 让 AI 持续运行 —— 运维思维与 AI 时代生存指南 | 物业管理 |

## 与 01 课程的区别

| 维度 | 01-工程实战 | 02-通识入门（本课程）|
|------|------------|---------------------|
| 受众 | 有 Python 经验的工程师 | 零编程基础 |
| 目标 | 会写代码 | 会理解、会判断、会提需求 |
| 代码 | 核心：动手写 | 辅助：看懂在做什么 |
| 练习 | 编码练习 | 讨论 / 方案设计 / 角色扮演 |
