# ❓ 为什么需要"多层"？

> 有一个"紧急停止"按钮不就够了？

```
❌ 只有安全带
   → 严重碰撞保不了命
```

**每一层保护不同级别的风险**
**多层叠加 = 全面防护**

# 🏥 风险评分 = 体检报告

```
你的体检报告：

  血压  120/80  → 🟢 正常
  血糖  6.8     → 🟡 偏高，注意
  胆固醇 280    → 🔴 危险，就医！

AI系统也有"体检报告"：

  风险评分：0-100分
  越高越危险
  
  📉 回撤程度  (40分)  从最高点跌了多少
  💸 日亏损    (30分)  今天亏了多少
  🔄 交易频率  (20分)  今天交易几次
  📦 持仓比例  (10分)  多少资金在仓位里
```

---

<!-- P17 -->
## 页 17/32 [TABLE]

# 四项"体检指标"

| 指标 | 权重 | 🟢 健康 | 🟡 注意 | 🔴 危险 |
|------|------|---------|---------|---------|
| 📉 回撤 | 40分 | < 3% | 3-7% | > 7% |
| 💸 日亏损 | 30分 | 盈利/微亏 | 达50%限额 | 接近限额 |
| 🔄 频率 | 20分 | < 2次 | 3-4次 | 5次满额 |
| 📦 持仓 | 10分 | < 30% | 30-70% | > 70% |

**评分解读：**
- 0-30分 🟢 一切正常
- 30-70分 🟡 密切观察
- 70-100分 🔴 立即干预

## 页 25/32 [CONCEPT]

# 🏆 法则一：监控 > 策略

```
  策略 = "怎么做"
    AI怎么判断该买还是该卖
    → 可能失效

  监控 = "做得怎么样"
    赚了多少、亏了多少、风险多高
    → 永远有用

  🚗 类比：
    导航路线（策略）可能因堵车改变
    但仪表盘（监控）永远在告诉你
    速度、油量、发动机状态
```

**策略在变，监控不变**

---

<!-- P26 -->
## 页 26/32 [CONCEPT]

# 🏆 法则二：必须有熔断

```
  熔断 = 家里的保险丝

  电流过大 → 自动断电 → 防止火灾

  你可能一辈子都不用保险丝
  但它必须在那里！

  因为：
    代码可能有Bug
    市场可能出现极端情况
    数据源可能出错
    第三方服务可能宕机

  你无法预测所有问题
  但可以确保：出错时系统自动保护
```

**不是"最好有" → 是"必须有"**

## 页 27/32 [CONCEPT]

# 🏆 法则三：过去 ≠ 未来

```
  AI从历史数据中学规律
  但过去的规律未必在未来成立

  🌤️ 就像天气预报：
     100年的数据能预测明天
     但预测不了从未发生过的灾害

  所以：
    ❌ 不能"设好了就不管"
    ✅ 持续监控表现
    ✅ 策略失效时及时调整或停止
```
