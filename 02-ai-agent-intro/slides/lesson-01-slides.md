# 看代码猜功能：ClawBot v1

| 代码 | 白话翻译 |
|------|---------|
| `self.memory_file = "memory.json"` | 用一个文件存对话记录 |
| `self.conversations = self._load_memory()` | 开机时把老对话读出来 |
| `history = self.conversations.get(user_id, [])` | 找到这个用户的聊天记录 |
| `response = openai.ChatCompletion.create(...)` | 把消息发给GPT-4，让它回复 |
| `self.conversations[user_id] = history[-20:]` | ⚠️ **只保留最近20轮！** |

## 页 12/30 [QUESTION]

# ❓ 关键一行

```python
history[-20:]
```

## 这行代码意味着什么？

**只保留最近20轮对话，更早的 → 直接删除！**

💬 如果你的秘书只有一个20页的笔记本……
写满了就把前面的**撕掉**……

## 页 14/30 [QUOTE]

# 💡

> **"有对话记录" ≠ "有记忆"**
>
> 对话记录 = 按时间排列的流水账
> 记忆 = 经过筛选、理解、分类的知识
>
> 人脑不会原样存储每一秒的经历
> **AI也不应该**

## 页 18/30 [TABLE]

# 什么信息放哪层？

| 你说的话 | 放哪层？ | 为什么？ |
|---------|---------|---------|
| "帮我查明天天气" | 工作记忆 | 查完就完了 |
| "今天和A公司开了会" | 短期记忆 | 今天的事 |
| "我对花生过敏" | **长期记忆** | 永久重要 |
| "报销流程：填表→审批→打款" | **知识库** | 通用规则 |

> 分层 → AI知道去**哪里找**信息
> 不再大海捞针
