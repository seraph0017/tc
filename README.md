## 统一目录结构

每个课程项目遵循相同的目录规范：

```
XX-课程名称/
├── README.md          # 课程说明
├── course-design.md   # 课程设计方案
├── origin/            # 原始素材（只读）
├── handout/           # 课程讲义（完整知识参考）
├── script/            # 讲课稿（口语化逐字稿 + 时间节奏）
├── slides/            # 课件内容脚本（Markdown 格式）
└── ppt/               # 演示文稿（.pptx）
```

## 三大教学载体

| 载体 | 定位 | 使用场景 |
|------|------|----------|
| **讲义** (handout) | 可信赖的参考书 | 课前预习、课后复习 |
| **讲课稿** (script) | 有温度的导航仪 | 控制教学节奏与互动 |
| **课件** (slides/ppt) | 视觉焦点引导器 | 课堂投屏展示 |
