# AI Agent 系统工程实战 — 从零到生产级多 Agent 架构

> 基于真实项目《从 ClawBot 到 OpenClaw：一个 AI Agent 系统的演进之路》，系统讲授 AI Agent 的设计、开发与运维全流程。

## 课程概览

- **课程性质：** 中高级技术实战课程
- **总课时：** 6 次课 × 3 小时 = 18 学时
- **目标受众：** 有 1-3 年 Python 开发经验的后端/全栈工程师

## 课程大纲

| 课次 | 主题 | 难度 |
|------|------|------|
| 第 1 课 | 从聊天机器人到智能助手：AI Agent 的记忆觉醒 | 中级 |
| 第 2 课 | 一个模型不够用：多模型调度与智能路由 | 中级 |
| 第 3 课 | 从独行侠到团队：多 Agent 架构设计与实现 | 中高级 |
| 第 4 课 | 让 Agent 写代码：结构化代码生成工作流 | 中级 |
| 第 5 课 | 实战案例：AI 驱动的量化交易系统（选修/进阶） | 高级 |
| 第 6 课 | 让系统活着：AI Agent 的工程化运维与演进复盘 | 中高级 |

## 项目结构

```
├── origin/          # 原始素材（只读）
├── course-design.md # 课程设计总方案
├── handout/         # 课程讲义（完整知识 + 可运行代码 + 练习）
├── script/          # 讲课稿（口语化讲解 + 时间节奏 + 互动标记）
├── slides/          # 课件内容脚本（每页一个核心信息，Markdown 格式）
└── ppt/             # 演示文稿（.pptx）
```

## 三大教学载体

| 载体 | 定位 | 信息深度 |
|------|------|----------|
| **讲义** (handout) | 可信赖的参考书 — 课前预习、课后复习 | ★★★★★ |
| **讲课稿** (script) | 有温度的导航仪 — 控制教学节奏与互动 | ★★★☆☆ |
| **课件** (slides) | 视觉焦点引导器 — 每页只传达一个核心信息 | ★☆☆☆☆ |

## 核心学习目标

完成本课程后，学员应能够：

1. 设计并实现完整的 AI Agent 记忆系统（工作记忆 / 短期 / 长期 / 知识库）
2. 构建多模型调度路由，根据任务类型自动选择最优模型
3. 设计并实现多 Agent 协作架构（通信、任务编排、记忆同步）
4. 运用 Skill 系统构建结构化的代码生成工作流
5. 建立 AI 系统的工程化运维体系（配置管理、监控报警、定时任务）
